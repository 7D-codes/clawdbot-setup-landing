**Files Created:**
```
infra/
├── terraform/
│   ├── main.tf              # VPS provisioning
│   └── cloud-init.yaml      # Node setup
├── api/
│   └── assignment-api.js    # User → instance assignment
├── scripts/
│   ├── pool-manager.js      # Idle pool maintenance
│   └── deploy.sh            # One-click deploy
├── caddy/
│   └── Caddyfile            # Wildcard SSL routing
└── package.json
```

**How it works:**
1. Terraform creates VPS nodes on Hetzner
2. Cloud-init installs Docker, Caddy, Node.js
3. Pool Manager keeps 5 idle OpenClaw containers ready
4. User signs up → Assignment API picks idle container → injects config → assigns subdomain
5. Caddy routes `user.clawdeploy.com` → container IP
